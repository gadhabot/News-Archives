Get news from any day since 2020-10-29

Architecture

Everyday 10:00 am GMT, thirty-four of the top trending articles from BBC News World is logged into a database. When you make a request to the API webpage, you recieve articles from a specified time frame or including a specified keyword.

Endpoints

  • /keyword
  • /day
  • /month
  • /year
